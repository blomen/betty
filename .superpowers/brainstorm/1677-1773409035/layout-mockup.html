Layout Breakdown

  1. Header bar — Symbol, price, change %, Compute/Scan buttons, threshold
  2. Indicator strip — Auto-computed pills: Regime, VIX, Market Type, Day Type (M7), Opening Type, IB, COT, RF
  3. Orderflow strip — Live signal badges: Delta, Aligned, Divergence, VSA, Trapped, TickVol, StopRun, CVD, Big Trades, Imbalance
  4. VP Levels grid — Session/Weekly/Leg/Macro profiles with POC/VAH/VAL + structural levels (PDH, VWAP, IB, OB, FVG)
  5. Signal table — Always visible, sorted by score. Each row shows setup name, level context, direction, entry/stop/target, R:R
