Classification Logic

Bet Classification
On loss:
  CLV positive → expected_loss (had real edge, variance)
  CLV negative → edge_erosion (market moved against)
  No CLV + edge < 1% → false_edge (thin/no edge)
  Kelly ratio > 1.5 → sizing_error (oversized)

On win:
  CLV positive → expected_win (real edge, deserved)
  CLV negative → bonus_win (got lucky, no real edge)
Trade Classification
On loss:
  R ≥ -1.0 → expected_loss (stopped out at plan)
  R < -1.0 + stop moved → stop_too_wide
  Hit stop but reversed → stop_too_tight
  Setup avg R < 0 → thesis_invalid

On win:
  R ≥ 2.0 → runner (let it ride)
  R > 0 → expected_win
