Trading Intraday Page — Redesign Approach

Drop manual gates, show everything as auto-computed indicators (like the betting page)

A

Indicator Dashboard + Signal Table

Top strip: Auto-computed indicator pills (Macro regime, Market type, Day type, Orderflow signals) — all visible, no gates. Color-coded green/yellow/red.

Middle: Multi-timeframe VP levels bar — session POC/VAH/VAL, weekly composite, leg profile, macro profile. Shows where current price sits relative to each VP anchor. User can input custom level anchors (leg start date, macro start date).

Bottom: Signal table always visible (no gating). Each signal row shows its score + which indicators confirm it.

Like ValuePage: FilterBar at top, signals always shown, indicators inline per row.

B

Level Map Focus + Sidebar Indicators

Left panel: Visual level map — price ladder showing all VP levels (session, weekly, macro), structural levels (PDH/PDL, OB, FVG), VWAP bands. Price position highlighted.

Right sidebar: Stacked indicator cards (Macro, Structure, Orderflow) — auto-computed, display-only, no checkboxes. Each card shows its signals with green/red/neutral state.

Bottom: Signal table, always visible.

More visual/spatial, but more complex to build. Price ladder is a significant component.

C

Hybrid: Indicator Strip + Collapsible Level Map

Top: Compact indicator strip (like A) — regime pill, market type pill, day type pill, orderflow signal badges. All auto, no gates.

Collapsible section: Multi-timeframe VP level map — shows POC/VAH/VAL for each profile as horizontal bars with labels. User sets anchor dates for leg/macro profiles via simple date inputs.

Main: Signal table always visible. Expanded rows show condition breakdown + level context.

Best of both: quick glance at indicators + detailed level view when needed. Closest to existing UI structure.

Common to all approaches

  • No Layer A / Layer B gates — everything auto-computed and displayed
  • No checkboxes — indicators are display-only (green/yellow/red state)
  • Signals always visible — no gating, just sorted by score
  • Multi-timeframe VP — session, weekly composite, leg, macro profiles
  • User inputs: only anchor dates for leg/macro VP start (or auto-detect swing points)
  • Macro data renamed from "bull/bear" to "risk on/off/mixed" (regime)
