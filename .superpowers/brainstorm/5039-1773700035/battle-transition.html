How the transition works

  1. Price approaches level → status changes to NEAR in table
  2. Price touches level → sound plays, layout morphs to battle mode
  3. Level table compresses to a minimal price strip at the bottom
  4. Gauges fill the main area with real-time orderflow + structure readings
  5. Trade action bar shows computed Entry/Stop/Targets + trade button
  6. Dismiss returns to scanning layout
